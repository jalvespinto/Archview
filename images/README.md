# ArchView Images

This directory contains visual assets for the ArchView extension.

## Required Images

### icon.png
- **Size**: 128x128 pixels
- **Format**: PNG with transparency
- **Purpose**: Extension marketplace icon
- **Design**: Should represent architecture diagrams and AI
- **Suggested elements**: Network/graph nodes, AI brain/circuit patterns, clean modern design

### banner.png
- **Size**: 1280x640 pixels (recommended)
- **Format**: PNG
- **Purpose**: Marketplace banner/header
- **Design**: Should showcase the extension in action
- **Suggested content**: Screenshot of an interactive architecture diagram with highlighted features

## Screenshot Guidelines

For marketplace screenshots (to be added to README.md):

1. **Main Feature Screenshot** (1920x1080)
   - Show a complete architecture diagram
   - Include the diagram toolbar and controls
   - Highlight an interactive element

2. **File Highlighting Screenshot** (1920x1080)
   - Show diagram with selected element
   - Show file explorer with highlighted files
   - Demonstrate IDE integration

3. **Abstraction Levels Screenshot** (1920x1080)
   - Show the same project at different abstraction levels
   - Side-by-side comparison if possible

4. **Multi-Language Screenshot** (1920x1080)
   - Show a polyglot project diagram
   - Highlight language-specific visual distinctions

5. **Export Dialog Screenshot** (1280x720)
   - Show the export functionality
   - Display export format options

## Design Guidelines

- Use the extension's color scheme (dark theme: #1e1e1e background)
- Ensure text is readable at small sizes
- Use high-quality, crisp images
- Avoid cluttered or busy designs
- Maintain consistency across all images
